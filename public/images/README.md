# Images Directory

Place your Rollermax images in this directory. The website is configured to use:

1. **camel.png** - Camel caravan image for traditional delivery showcase
2. **Motorcycle.png** - Motorcycle courier for urban delivery
3. **Aeroplane.png** - Cargo airplane for international shipping
4. **lorry.png** - Cargo truck for heavy freight transport
5. **Banner.png** - Rollermax banner for brand showcase
6. **Rollermax logo.png** - Official logo (used in Logo3D component)

## Image Usage

These images are automatically displayed in:
- ImageGallery component (homepage)
- Various sections showcasing delivery methods

## Image Format

- Format: PNG or JPG
- Recommended size: 1200x800px or larger
- Optimize for web before uploading
